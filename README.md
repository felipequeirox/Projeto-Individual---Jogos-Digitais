# Projeto-Individual---Jogos-Digitais

CoinRush: Wanted é um minigame 2D desenvolvido em Unity como projeto acadêmico. O jogador deve coletar 4 moedas espalhadas pelo mapa em até 15 segundos, desviando de policiais que causam derrota imediata ao contato.

Como jogar:
Mova o personagem com WASD ou setas do teclado. Suporte a controle Xbox pelo analógico esquerdo. Colete todas as moedas antes do tempo acabar sem encostar nos inimigos.
